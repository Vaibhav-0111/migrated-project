📌 ONE-LINE PITCH
================================================================================

Saesha is an ADHD-friendly, AI-powered interactive math learning platform
that transforms abstract CBSE Class 11–12 mathematics into calm, game-based
3D experiences — with adaptive difficulty, real-time narration, and an AI tutor.

================================================================================
🎯 WHAT IT IS
================================================================================

Saesha (meaning "meaningful life" in Sanskrit) is a full-stack web application
designed for neurodivergent learners — especially students with ADHD — who
struggle with traditional math education. Instead of textbooks and lectures,
students learn through 15 interactive 3D game scenes across 5 math chapters,
each with AI-generated quizzes, step-by-step helpers, voice narration, and a
built-in AI chatbot tutor.

Key Philosophy: "Intuition before formulas"
→ Students FEEL the math through visual interaction BEFORE encountering
  symbolic notation or memorization.

================================================================================
🧩 THE PROBLEM IT SOLVES
================================================================================

1. 🧠 ADHD & Learning Differences: Traditional math platforms use timers,
   dense text, and high-pressure formats that trigger anxiety. Saesha uses
   calm aesthetics, no timers, bite-sized sessions (3–10 min), and gentle
   animations.

2. 📉 Math Anxiety: Abstract formulas without context lead to disengagement.
   Every concept is introduced through a real-world scenario (shadows for
   trigonometry, gift-wrapping for surface area, rescue missions for sine).

3. 📚 One-Size-Fits-All: Traditional platforms don't adapt. Saesha uses
   Exponential Moving Average (EMA) algorithms to adjust difficulty in
   real-time based on accuracy, response time, and hint usage.

4. 🎮 Engagement Gap: Dry question banks bore students. Saesha gamifies
   learning with 17 earnable badges, star ratings, streak tracking, and
   a leaderboard.

🤖 AI FEATURES
================================================================================

1. 🧠 AI Question Generation (Serverless Edge Function)
   - Model: Google Gemini 2.5 Flash (via Lovable AI Gateway)
   - Generates 8 unique, grade-specific (11 or 12) MCQs per game session
   - CBSE-aligned, difficulty-aware (easy/medium/hard)
   - LaTeX-formatted math expressions rendered with KaTeX
   - Fallback: Curated static question banks if AI unavailable

2. 💬 AI Chatbot Tutor ("Saesha")
   - Model: Google Gemini 3 Flash Preview (streaming)
   - Real-time streaming responses via Server-Sent Events (SSE)
   - Personality: Friendly, encouraging, step-by-step math guidance
   - LaTeX rendering in chat for inline math formulas
   - Rate limiting and credit-aware error handling

3. 🔊 Text-to-Speech Narration
   - API: ElevenLabs Turbo v2.5
   - Voice: Natural human voice narration for tutorials and explanations
   - MP3 audio streaming to browser

4. 📈 Adaptive Difficulty Engine
   - Uses Exponential Moving Average (EMA) algorithm
   - Tracks: accuracy, response time, hints used
   - Auto-adjusts difficulty level (1–5) per subject per student
   - Implemented in the submit-result serverless function

🗃️ DATABASE SCHEMA (PostgreSQL)
================================================================================

9 Tables:

  1. profiles — User profile data (name, grade, subjects)
  2. games — Game definitions (15 games, chapter, concept, grade)
  3. game_progress — Per-user per-game per-difficulty progress
  4. questions — AI-generated & static question bank
  5. sessions — Learning session tracking
  6. task_results — Individual answer records
  7. user_performance — EMA-based adaptive performance metrics
  8. rewards — Earned badges & achievements
  9. analytics_events — Event logging for learning analytics

Database Functions:
  - handle_new_user() — Auto-creates profile on signup (trigger)
  - unlock_next_level() — Auto-unlocks next difficulty on completion (trigger)
  - unlock_first_games_for_user() — Unlocks all easy levels for new users
  - increment_session_stats() — Atomic session counter (with ownership check)
  - generate_questions_for_game() — Placeholder question seeder

================================================================================
☁️ SERVERLESS ARCHITECTURE (Edge Functions)
================================================================================

8 Deno Edge Functions deployed on Lovable Cloud:

  1. generate-question — AI question generation (Gemini 2.5 Flash)
     Input: chapter, gameTitle, difficulty, grade
     Output: MCQ with question, options, explanation, hint

  2. chat — AI tutor chatbot (Gemini 3 Flash, streaming SSE)
     Input: message history
     Output: Streamed AI response with LaTeX math

  3. text-to-speech — Voice narration (ElevenLabs)
     Input: text, optional voiceId
     Output: MP3 audio buffer

  4. submit-result — Answer submission + adaptive engine
     Input: answer data (correct, time, hints)
     Output: updated difficulty, tokens, streak

  5. award-badge — Server-validated badge awarding
     Input: badgeId
     Output: success/already earned/requirements not met

  6. get-games — Fetch game catalog
  7. get-user-stats — Fetch aggregated user statistics
  8. update-game-progress — Update game completion data
  9. seed-questions — Admin: populate question bank

All functions: JWT-authenticated, CORS-enabled, error-handled.
